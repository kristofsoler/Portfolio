Understanding the Socio-Economic Dynamics of Political Affiliation in Norway Using SHAP: Insights from European Social Survey 10

The above tables show the output variable scale and the top 8 features that were identified using SHAP.

Introduction

This project investigates the socio-economic factors influencing political affiliation in Norway, using data from the European Social Survey (ESS) 10. The study employs machine learning models, with the XGBoost regressor achieving the best performance in predicting political leaning on a 0–10 left-right scale. Using SHAP (SHapley Additive exPlanations), the analysis reveals key features driving political attitudes, including income inequality, immigration, and opinions on European Union unification

Methods

The project used Python for data preprocessing, model training, and interpretability analysis. Missing data in the ESS dataset was handled using bootstrapping, which resampled available data to preserve dataset integrity and ensure robust analysis. To standardize the data, a StandardScaler was applied, transforming categorical variables onto consistent scales suitable for machine learning algorithms. Non-parametric models were chosen due to the lack of Gaussian distribution in categorical data, and the dataset was split into training and testing sets for model evaluation. Several machine learning models, including Random Forest, K-Nearest Neighbors (KNN), and XGBoost, were trained and evaluated using performance metrics such as Mean Squared Error (MSE), Mean Absolute Error (MAE), and R². Hyperparameter tuning was performed using grid search with cross-validation, ensuring the optimal configuration of the models. The XGBoost regressor outperformed others, demonstrating lower MSE and higher R² values on both training and testing datasets.

Key Findings

The findings highlight that left-leaning voters prioritize reducing income inequality and supporting immigration, while right-leaning voters oppose income redistribution and prefer limited immigration. Interestingly, enthusiasm for European Union unification aligns more with right-wing voters, contrary to broader trends. These insights provide a nuanced understanding of the socio-economic factors shaping political polarization in Norway.
